# CS-370-11971-M01-Current-Emerging-Trends-in-CS-1-
Current/Emerging Trends in CS 
For this project, I implemented a reinforcement learning–based pirate intelligent agent using a Jupyter Notebook. I was provided with starter code that included the project structure, environment setup, and some of the core functions for the pirate world—such as the environment class, state representation, and reward definitions. These components established the “rules” of the world that the agent had to navigate.
The code I created myself focused on the learning algorithm and the agent logic. This included:

Implementing or completing the Q‑learning algorithm.
Designing the training loop to allow the agent to interact with the environment.
Tuning hyperparameters such as learning rate, discount factor, and exploration rate.
Adding visualizations and debugging outputs to understand how the agent improved over time.
Testing and evaluating the agent’s performance after training.

Through this process, I transformed the provided environment into a functioning intelligent agent capable of making decisions, learning from experience, and optimizing its behavior.
What do computer scientists do, and why does it matter?
Computer scientists solve problems using computation. They design algorithms, develop intelligent systems, organize and analyze data, and build software that powers modern life. Their work matters because nearly every industry—healthcare, finance, transportation, entertainment, and more—depends on computational tools. In this project, using reinforcement learning to teach an agent how to navigate an environment is a small example of what computer scientists do: turning abstract theories and models into practical, working solutions.

How do I approach a problem as a computer scientist?
A computer scientist approaches problems systematically:

Understand the problem and constraints. For this project, that meant studying the pirate environment, actions, and rewards.
Break the problem into smaller components. I separated environment logic, learning algorithm, training, and evaluation.
Choose appropriate tools and techniques. Reinforcement learning, specifically Q‑learning, was the right fit for a decision‑making problem involving exploration and rewards.
Iterate and experiment. I adjusted parameters, monitored results, and refined the agent’s behavior.
Evaluate and reflect. I examined how well the agent learned and what factors influenced its performance.

This structured approach helps ensure that solutions are not only functional but also efficient, understandable, and adaptable.

What are my ethical responsibilities to the end user and the organization?
Computer scientists have significant ethical responsibilities because their work impacts real people. Some of these responsibilities include:

Ensuring safety and reliability. Systems should behave predictably and avoid harming users.
Protecting privacy and data. Any data collected or processed must be handled responsibly.
Avoiding bias in algorithms. Machine learning models must be designed and tested to avoid unfair or discriminatory behavior.
Being transparent about limitations. Users and organizations should understand what a system can and cannot do.
Considering long‑term consequences. Technologies should benefit society rather than create harm or inequity.

Even though this project was a simplified simulation, the same principles apply. When building intelligent agents—even pirate-themed ones—computer scientists must think about how decisions are made, how reward structures shape behavior, and what unintended outcomes might emerge.
